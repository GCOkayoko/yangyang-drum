# 大鼓 大鼓 敲敲敲

秧秧敲鼓互动小页面——点击或按键，看秧秧敲鼓！

## 运行方式

直接用浏览器打开 `index.html`，或运行 Python 版本：

```bash
python 秧秧敲鼓.py
```

## 操作说明

| 操作 | 效果 |
|------|------|
| 点击画面 | 敲一下鼓 |
| 键盘 F / J | 敲一下鼓 |
| 空格 | 启停自动播放 |
